Hexo+Next+Github 搭建博客

Posted on 2021-08-25

记录下使用Hexo博客框架，Next主题和Github搭建博客的过程。

搭建博客

安装 Hexo

前提已经安装了Node环境

初始化博客目录

运行博客

访问默认的地址http://localhost:4000 就可以看到博客页面了

安装 Next 主题

下载主题

访问https://github.com/iissnan/hexo-theme-next/releases 下载最新的release版本。

安装主题

解压文件后，重命名文件夹为next，放置在博客根目录/theme下。

同时修改/_config.yml，讲博客主题改为next。

注意

5.0+版本 的hexo，需要手动安装swig

部署至 Github
