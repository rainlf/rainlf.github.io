# Markdown 语法

## 标题
```
## 标题
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

## 水平分割线
```
---
***
```
---
***

## 文本
```
**粗体**，__粗体__
*斜体*，_斜体_
~~删除线~~
```
**粗体**，__粗体__
*斜体*，_斜体_
~~删除线~~

## 列表

```
-   一级列表

        *   三级列表
```

## 表格

### 左侧对齐
```
| Title1 | Title2 |
| ------ | ------ |
| foo    | bar    |
```
| Title1 | Title2 |
| ------ | ------ |
| foo    | bar    |

### 居中对齐
```
| Title1 | Title2 |
| :----: | :----: |
|  foo   |  bar   |
```
| Title1 | Title2 |
| :----: | :----: |
|  foo   |  bar   |

### 右侧对齐
```
| Title1 | Title2 |
| -----: | -----: |
|    foo |    bar |
```
| Title1 | Title2 |
| -----: | -----: |
|    foo |    bar |
