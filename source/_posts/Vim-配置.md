## 配置文件

```shell
vi ~/.vimrc
```

## 内容

```shell
syntax enable
set nu
set syntax=on
set showcmd
set autoread
set nobackup
set ruler
set cursorline
set magic
set noeb
set smartindent
set autoindent
set shiftwidth=4
set tabstop=4
set mouse=a
set hlsearch
filetype on
```
